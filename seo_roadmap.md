## ⚡ НЕДЕЛЯ 1 (3–10 июня 2026) — ТЕХНИЧЕСКИЕ КРИТИЧЕСКИЕ ИСПРАВЛЕНИЯ

Цель: устранить всё, что мешает поисковикам обходить и индексировать сайт.

**День 1-2 — robots.txt (P0)**
- [ ] Открыть /media/ — убрать `Disallow: /media/`
- [ ] Открыть /shop/ — убрать `Disallow: /shop/`
- [ ] Открыть /gold/ — убрать `Disallow: /gold/`
- [ ] Оставить закрытыми: /_nuxt/, /api/, /admin/, /cart/, /checkout/
- [ ] Перегенерировать sitemap.xml со всеми открытыми разделами
- [ ] Отправить новый sitemap в Google Search Console и Яндекс.Вебмастер

**День 1-2 — Canonical trailing slash (P0)**
- [ ] https://kupikod.com/console → 301 → /console/
- [ ] https://kupikod.com/gold → 301 → /gold/
- [ ] https://kupikod.com/shop/ → 301 → /shop
- [ ] https://kupikod.com/terms/ → 301 → /terms
- [ ] https://kupikod.com/shop/podarocnye-karty/steam/ → 301 → /steam

**День 3-4 — 500 ошибки (P0)**
- [ ] Проверить лог сервера по 96 страницам /media/*
- [ ] Исправить серверную причину (скорее всего — проблема с БД / CMS / деплоем)
- [ ] Отправить исправленные URL на переобход в GSC и Яндекс.Вебмастер

**День 4-5 — Missing H1 (P1)**
- [ ] https://kupikod.com/promocode → добавить H1: "Промокоды Kupikod — скидки на игры и подписки"
- [ ] https://kupikod.com/media/novosti → добавить H1: "Новости игровой индустрии и киберспорта"
- [ ] https://kupikod.com/media/stati → добавить H1: "Статьи об играх — обзоры, гайды, рейтинги"
- [ ] https://kupikod.com/media/promokody → добавить H1
- [ ] https://kupikod.com/media/tag/* (4 страницы) → добавить H1

**День 5-6 — 302 → 301 (P1)**
- [ ] Заменить 4 редиректа 302 на 301

**День 6-7 — Главная страница быстрые правки (P1)**
- [ ] Исправить Title: "Kupikod — Купить игры, подписки, сделать донат | Цифровой маркетплейс"
- [ ] Убрать число из H1 если есть
- [ ] Добавить минимум 300 слов контента на главную

**Ожидаемый эффект за неделю 1:**
- Открытие robots.txt → тысячи страниц начнут нормально обходиться
- Устранение 500 ошибок → возврат статей в индекс
- Canonical fixes → устранение дублей

## 📈 НЕДЕЛИ 2–4 (10 июня — 1 июля 2026) — КОММЕРЧЕСКИЕ СТРАНИЦЫ

Цель: восстановить 404, усилить ключевые коммерческие страницы.

**Восстановление 404 — игровые лендинги (P0)**
- [ ] /shop/fortnite-v-bucks или /fortnite → создать страницу донат Fortnite V-Bucks
- [ ] /gold/pubg или /shop/pubg-uc → создать страницу UC PUBG (одна из топ-позиций Яндекс)
- [ ] /shop/minecraft → создать страницу ключа Minecraft
- [ ] /shop/robux-roblox → создать страницу Robux Roblox (особенно важно: молодёжная ЦА)
- [ ] /shop/valorant-points → страница Valorant Points
- [ ] /shop/brawl-stars → страница доната Brawl Stars

Для каждой страницы: уникальный title, H1, 300+ слов описания, FAQ (5-7 вопросов), цены, инструкция.

**Каннибализация — /media/tag/* (P1)**
- [ ] /media/tag/novosti → canonical на /media/novosti
- [ ] /media/tag/stati → canonical на /media/stati
- [ ] /media/tag/promokody → canonical на /media/promokody
- [ ] /media/tag/interaktivnye-karty → canonical на /media/interaktivnye-karty

**Устранение "-articlepapa" дублей (P1)**
- [ ] /media/*-articlepapa → 301 → оригинальная статья

**Title/Description оптимизация — ключевые страницы (P1)**
Приоритетный список:
- [ ] /shop/igry — убрать "7991" из H1
- [ ] /shop/igrovaia-valiuta — убрать "102" из H1
- [ ] /shop/podpiski — убрать "21" из H1
- [ ] /shop/podarocnye-karty/steam — убрать "216" из H1
- [ ] /shop/catalog — исправить title "Каталог — купить на kupikod.com"
- [ ] /gold/ — усилить H1: "Донат в мобильные игры — быстрое и безопасное пополнение"

**Перелинковка — устранить дисбаланс (P1)**
- [ ] Добавить ссылку на /gold/ из главной страницы в Hero-блок
- [ ] Добавить /gold/ в футер сайта
- [ ] Добавить /promocode в хедер или навигацию
- [ ] Добавить ссылку /console/ → /shop/podpiski и обратно

**Ожидаемый эффект:**
- Восстановление позиций по Fortnite, PUBG, Minecraft, Roblox
- Устранение каннибализации в разделе /media/
- Улучшение CTR по ключевым коммерческим страницам

## 🚀 МЕСЯЦ 2–3 (Июль — Август 2026) — КОНТЕНТ И СТРУКТУРА

Цель: усилить существующие страницы, создать информационные посадочные.

**Расширение контента на коммерческих страницах (P1)**
- [ ] /console/ — добавить FAQ (8-10 вопросов), отзывы, блок "Как это работает"
- [ ] /gold/ — расширить до 1 500+ слов: список игр, инструкция, FAQ
- [ ] /shop/podarocnye-karty/steam — добавить инструкцию активации, FAQ
- [ ] /shop/podpiski — добавить сравнение подписок, инструкции по каждой
- [ ] / (главная) — добавить 500+ слов: USP, преимущества, популярные категории, FAQ

**Информационный контент — новые статьи (P1)**
- [ ] "Как пополнить PS Store из России в 2026 году" → перелинковка на /console/
- [ ] "Как купить Fortnite V-Bucks в России" → на новую страницу товара
- [ ] "Топ 20 игр для ПК в 2026 году" → на /shop/igry
- [ ] "Как сделать донат в PUBG из России" → на /gold/pubg
- [ ] "Как купить Minecraft в России в 2026" → на /shop/minecraft

**Title/Description массовая оптимизация (P2)**
- [ ] Разработать шаблоны для:
  - Товарные страницы /shop/: "Купить [название] — [регион] | от [цена] | Kupikod"
  - Категории: "Купить [категория] — лицензионные ключи и карты | Kupikod"
  - Статьи: "[Тема статьи] | Kupikod Media"
- [ ] Применить шаблоны программно ко всем 3 854 длинным title

**Микроразметка Schema.org (P2)**
- [ ] Добавить Product + Offer на товарные страницы /shop/
- [ ] Добавить FAQPage на коммерческие разделы
- [ ] Добавить Article на /media/ статьи
- [ ] Добавить BreadcrumbList на все страницы > depth 2

**Скорость сервера (P2)**
- [ ] Аудит серверной конфигурации (239 страниц > 5s)
- [ ] Внедрить кэширование на уровне Nginx/Redis
- [ ] CDN для статики (изображения, JS, CSS)
- [ ] Оптимизировать Nuxt.js SSR/SSG
- [ ] Целевые показатели: avg < 0.5s, max < 3s

**Ожидаемый эффект:**
- Рост позиций по информационным запросам (топ-10 Google)
- Улучшение поведенческих факторов (время на сайте, глубина)
- Rich Snippets в выдаче Google

## 📊 МЕСЯЦ 3–6 (Сентябрь — Декабрь 2026) — МАСШТАБИРОВАНИЕ

Цель: масштабировать видимость, расширить семантику, усилить ссылочный профиль.

**Расширение игровых лендингов (P1)**
- [ ] Создать страницы для всех 404 игр из списка (30+ страниц)
- [ ] Приоритет: clash-of-clans, mobile-legends, free-fire, zenless-zone-zero, honkai, genshin (отдельные подстраницы валюты)
- [ ] Для каждой игры: донат-страница + связанные статьи

**Расширение /media/ (информационный кластер) (P2)**
- [ ] 20+ статей в месяц по темам: гайды, системные требования, топ-листы, новости
- [ ] Кластеризация: каждая статья → ссылка на коммерческую страницу
- [ ] Фокус на запросы из видимости Яндекс (из analiz.pdf): Standoff 2, Dead by Daylight, Dying Light, Call of Duty, Far Cry

**Внутренняя перелинковка — системная (P2)**
- [ ] Аудит всех 558 страниц с 1-3 inlinks
- [ ] Добавить блок "Похожие игры" / "Купить также" на товарных страницах
- [ ] Добавить блок "Статьи по теме" на страницах игр
- [ ] Автоматические блоки "Популярные в разделе" на категорийных страницах

**Ссылочный профиль (P2)**
- [ ] Текущий TF/CF = 16/10 — есть потенциал роста
- [ ] Целевые TF ≥ 25 к концу года
- [ ] Приоритет: игровые медиа (DTF, StopGame, 4pda), YouTube описания, Telegram-каналы геймеров
- [ ] Не покупать ссылки — естественное наращивание (обзоры, упоминания)

**E-E-A-T усиление (P2)**
- [ ] Создать страницу /about (кто мы, миссия)
- [ ] Создать страницу /reviews (отзывы покупателей — интеграция с отзовиками)
- [ ] Добавить авторов к статьям /media/ (фото, bio)
- [ ] Страница /safety или /guarantee (гарантии безопасности)

**ORM (Репутация) (P3)**
- [ ] Мониторинг упоминаний "купикод отзывы" в поиске
- [ ] Работа с отрицательными отзывами
- [ ] Размещение карточки компании на Яндекс.Картах, 2GIS
